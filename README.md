## Install 
1. Install chromedriver [here](https://chromedriver.chromium.org/)
2. `pip install -r requirements.txt`
3. `virtualenv venv`
4. `source venv/bin/activate`
5. fill in credentials in secret.py
6. `python tinder_bot.py`
